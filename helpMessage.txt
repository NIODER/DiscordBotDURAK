Информация по командам и работе бота.
Команды, доступные всем:
	$random - выдает случайное число
	реши - отвечает да/нет
	$ID - отправляет в лс твой id
	$surf - скидывает адрес серф сервера для кс
Команды, доступные только админам:
	$set + #названиеканала + тип канала 
		0 - бот не модерирует канал,
		1 - канал "welcome" (в котором появляются сообщения о новичках на сервере)
		2 - канал для ссылок в сеть
		3 - канал для флуда (все каналы имеют тип 3 по умолчанию)
	$admin + упоминание - выдает продвинутый доступ к боту
	$spam + число + текст - спамит указанный текст указанное количество раз
	$moderate - модерирует последние 100 сообщений
	$clean + текст - удалает все сообщения с указанным текстом
Бот отвечает на сообщения, содержащие ряд триггеров, например "?"
