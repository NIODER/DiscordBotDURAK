Я выделяю 4 вида текстовых каналов: невидимый для бота канал, 
приветственный канал, флуд, и канал для ссылок.
По порядку, флуд - бот модерирует сообщения на предмет ссылок или команд
ссылки - бот пересылает сюда ссылки
приветственный канал можно не вводить, если не хочешь, чтобы приветственные
сообщения не удалялись. Невидимы канал бот не модерирует, он туда не лезет и там не работает.

Вот ряд команд, который нужно будет прописать:
$set + #названиеканала + тип канала 
		0 - бот не модерирует канал,
		1 - канал "welcome" (в котором появляются сообщения о новичках на сервере)
		2 - канал для ссылок в сеть
		3 - канал для флуда (все каналы имеют тип 3 по умолчанию)
Пример: $set #хуи-и-залупы 2 - мы обозначили канал хуи-и-залупы как 
канал для ссылок. (Канал для работы с другими ботами, например музкальными, лучше делать
с типом 0)

$admin + упоминание - выдает продвинутый доступ к боту.
Пример: $admin @DURAK 

Для гайда по командам пиши $help
